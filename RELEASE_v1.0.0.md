## 📦 安装说明

### 便携版（推荐）
1. 下载 `XIAOHAI.AI-v1.0.0-portable.zip`
2. 解压到任意目录
3. 运行 `XIAOHAI.AI.exe`

### 系统要求
- Windows 10/11 x64
- .NET 10.0（已内置在便携版中）
- Ollama 本地模型服务（可选）
